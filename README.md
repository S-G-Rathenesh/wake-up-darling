# Wake Up Darling ❤️⏰

Wake Up Darling is a couple-based smart alarm application designed to strengthen relationships through interactive wake-up features and real-time communication.

## 🌟 Features

### 🔔 Smart Wake Requests
Send wake-up requests directly to your partner with instant notifications.

### 🚨 Emergency Wake Mode
Trigger a high-priority emergency alarm when urgent attention is needed.

### 💬 Real-Time Chat
Built-in private chat system for couples to communicate seamlessly.

### 🔥 Wake Streak Tracking
Track successful wake attempts and maintain streak records.

### 📊 Relationship Stats Dashboard
View total wake attempts, successful streaks, longest streak, and activity insights.

### 📱 Alarm Status Monitoring
Check whether your partner turned off, ignored, or responded to alarms.

### 🔕 Background Notification Support
Reliable background execution to ensure alarms and notifications work even when the app is closed.

### 🔐 Secure Authentication
Safe and secure login system for couples.
