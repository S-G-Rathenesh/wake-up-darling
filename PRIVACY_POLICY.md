## 1. Introduction

Wake Up Darling ("the App") is a couple-based alarm and communication application designed to help partners stay connected through shared alarms, messaging, and voice/video calling.

This Privacy Policy explains what information is collected and how it is used.

By using the App, you agree to this policy.

---

## 2. Information We Collect

### 2.1 Account Information
- Email address
- Unique User ID (UID)
- Collected via Firebase Authentication

### 2.2 Messages
- Text messages
- One-time images
- Media shared between paired partners
- Timestamps and delivery status

### 2.3 Alarm Data
- Alarm schedules
- Wake results
- Streak tracking data

### 2.4 Call Metadata
- Call type (voice/video)
- Duration
- Connection status

Voice and video streams are processed via ZegoCloud and are not recorded or stored by the App.

### 2.5 Device Information
- FCM device token (for notifications)
- Device timezone (for alarm accuracy)

---

## 3. Media Storage

- All media is stored in private local app storage.
- Media is NOT stored on Cloudinary or any external cloud.
- Media is NOT visible in device gallery.
- Media is deleted automatically when the app is uninstalled.
- One-time images are deleted after being viewed.
- Users can manually download normal media to gallery if desired.

---

## 4. Third-Party Services

| Service | Purpose |
|----------|----------|
| Firebase | Authentication, Database, Notifications |
| ZegoCloud | Voice & Video Calling |

These services operate under their own privacy policies.

---

## 5. Data Security

- All communication uses HTTPS/TLS encryption.
- Firestore security rules restrict access.
- Media stored in private app directory.
- No public indexing of data.

No method of transmission is 100% secure.
